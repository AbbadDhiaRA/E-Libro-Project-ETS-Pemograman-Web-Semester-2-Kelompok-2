# E-Libro-Project-ETS-Pemograman-Web-Semester-2-Kelompok-2
**E-Libro | Proyek ETS Pemrograman Web Semester 2 – Kelompok 2**

E-Libro merupakan platform e-commerce dan perpustakaan digital berbasis web yang dikembangkan sebagai proyek Ujian Tengah Semester (ETS) mata kuliah Pemrograman Web Semester 2 oleh Kelompok 2.

Website ini dirancang untuk menghadirkan pengalaman toko buku digital modern dengan berbagai fitur, antara lain:

* Homepage dengan promosi buku unggulan dan populer
* Fitur eksplorasi serta pencarian buku
* Sistem My Library pribadi
* Desain antarmuka modern, responsif, dan user-friendly

### Ketersediaan Konten Saat Ini:

Saat ini, buku-buku yang ditampilkan dalam website masih berfungsi sebagai konten placeholder/demo antarmuka dan belum memiliki file bacaan yang terintegrasi, karena file pdf buku yang terlalu besar untuk diupload ke repository.

### Tujuan Proyek:

Repository ini dikembangkan sebagai bentuk implementasi pembelajaran dalam:

* HTML
* CSS
* JavaScript
* Responsive Web Design
* Frontend Development
* Local Storage Integration

### Pengembangan Selanjutnya:

Beberapa potensi pengembangan di masa depan meliputi:

* Penambahan koleksi buku
* Integrasi lebih banyak file PDF
* Sistem backend dan database
* Autentikasi pengguna
* Pengembangan fitur e-commerce yang lebih lengkap

E-Libro merupakan representasi komitmen kami dalam menggabungkan kebutuhan akademik dengan standar pengembangan website yang profesional.
